LLM Execution Contract
$ export VYBE_AGENT=demo-agent
$ vybe ... --request-id demo_$RANDOM # every mutation
$ vybe resume --peek # read-only context

Pattern: mutate → verify readback. Use a fresh --request-id on every mutating call so retries are safe and duplicate-free. Hook telemetry is best-effort and fire-and-forget; core state writes remain durable in SQLite. Hook commands shown in this demo simulate lifecycle events that are normally auto-fired by integrations.
